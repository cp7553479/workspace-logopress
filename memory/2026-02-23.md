# 2026-02-23

## WeCom Channel Troubleshooting

**Issue:** WeCom agent channel was receiving messages but not responding.

**Root Cause:** Error 60020 - "not allow to access from your ip"
- Server IP: `43.163.200.29`
- This IP was not in the WeCom application's trusted IP whitelist

**WeCom Config:**
- CorpId: `ww1c18e276669faf9e`
- AgentId: `1000008`

**Solution:** Add server IP `43.163.200.29` to WeCom app's trusted IP list
- Go to: 企业微信管理后台 → 应用管理 → 自建应用 → 开发者接口 → 可信 IP
- Add the server IP and wait a few minutes for it to take effect

**Log Evidence:**
```
[wecom-outbound] Failed to send text: Error: send failed: 60020 not allow to access from your ip, hint: [...], from ip: 43.163.200.29
```
